# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-02-13

### Added
- Initial release
- Flask backend with Yahoo Finance integration
- Real-time stock data fetching
- Technical indicators: SMA 50, SMA 200, RSI 14
- Buy/Sell/Hold signal generation
- 30-day sparkline charts using Chart.js
- Responsive web UI with gradient design
- Google Colab support with ngrok tunnel
- One-command deployment script
- Comprehensive documentation

### Features
- Track up to 50 NASDAQ stocks
- Automatic indicator calculations
- Color-coded trend and signal display
- 52-week range tracking
- % distance from 52-week high
- Manual refresh functionality
- Error handling for invalid tickers
